우리 반 우주 - 학기 중/방학 중 모드 전환 버전

업로드 파일
- index.html
- admin.html
- functions/api/data.js

기능
1. 관리자 설정에서 학기 중 / 방학 중 모드 전환
2. 학기 중: 기존 4개 장소별 체크리스트, 70%부터 조각 지급
3. 방학 중: 장소 구분 없이 10개 항목을 한 화면에 표시, 50%부터 조각 지급
4. 학생의 조각과 별은 모드를 바꿔도 계속 누적
5. 과거 활동 기록에는 당시 운영 모드가 함께 저장

배포
GitHub 저장소에서 위 3개 파일을 같은 경로에 덮어쓰고 Commit합니다.
Cloudflare Pages가 자동으로 새 배포를 완료한 뒤 관리자 페이지의 설정 탭에서 모드를 변경합니다.
D1 바인딩 이름은 기존과 동일하게 DB를 사용합니다.
